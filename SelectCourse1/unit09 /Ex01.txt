public class Exercise1 {
   public static void main(String[] args) {

      for (int i = 0, j = 0; i < 10&&j< 10; i++, j++) {
        System.out.println(i + " + " + j + " = " + (i+j));
      }
    }
  }

런타임 에러
i,j는 for문안에 있는 지역변수인데 for문 밖에다가 씀/i < 10, j< 10;이것도 틀림

결과
0+0=0
1+1=2
2+2=4
3+3=6
...
9+9=18
